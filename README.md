  By using the build.zig to config to download repo from github. There are two types. One is repo git and other is binary dev build for debug.

  Sample project sample test.

  Note there are zig commands.
# zig commands:
```
zig fetch --save=sdl https://github.com/libsdl-org/SDL/releases/download/release-3.4.12/SDL3-devel-3.4.12-mingw.zip
```
  This download package name with the hash in zig-pkg.
